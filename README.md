## 🚀 How to Run

1. **Clone the repository:**

```bash
git clone https://github.com/abjaiyad/Python-Exercises.git
```

2. **Navigate to the folder of the exercise you want to run:**

```bash
cd "01_Variables"
```

3. **Run the Python program:**

```bash
python exercise1.py
```
